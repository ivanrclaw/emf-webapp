## Bugs Encontrados

### BUG-1: Botón "Open Template Library" no abre el modal (FIXED)
- **Severidad:** Media
- **Componente:** IDEToolbar / CodeGenIDE
- **Descripción:** El botón `onClick={onOpenLibrary}` no disparaba `setLibraryOpen(true)`. Causa: stale closure — el prop `onOpenLibrary={() => setLibraryOpen(true)}` se creaba como lambda inline en cada render, perdiendo la referencia.
- **Fix:** Usar `const openLibrary = useCallback(() => setLibraryOpen(true), [])` y pasarlo como prop estable.
- **Estado:** ✅ RESUELTO
